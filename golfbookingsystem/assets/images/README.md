Add user and trainer photos to this folder.

Example paths to enter in the app:

- `assets/images/user.jpg`
- `assets/images/nordin.jpg`
- `assets/images/muhayuddin.jpg`

Main-menu banner:

- Add a transparent PNG named `banner.png`.
- The app displays it from `assets/images/banner.png`.
- A golf icon placeholder appears until the file is added.

App logo:

- `logo_original.png` is the original supplied logo.
- `app_logo.png` is the cropped transparent copy used on login, the main banner, and app headers.
